После скачивания всех файлов из репозитория проведем инициализацию директории:

terraform init

Развернем инфраструктуру с помощью terraform:

terraform validate      # Проверит код на валидность 
terraform apply         # Покажет изменения и запустит сборку инфраструктуры

В output будут находится внешние и внутренние ip адреса для созданных машин.
